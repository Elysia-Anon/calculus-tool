考研高数必背公式

1. 基本积分公式

∫ x^n dx = x^(n+1)/(n+1) + C (n≠-1)

∫ 1/x dx = ln|x| + C

∫ e^x dx = e^x + C

∫ sin(x) dx = -cos(x) + C

∫ cos(x) dx = sin(x) + C

2. 常用导数

(C)' = 0

(x^n)' = n*x^(n-1)

(sin x)' = cos x

(cos x)' = sin x

(ln x)' = 1/x

3. 泰勒公式 (Maclaurin)

e^x = 1 + x + x^2/2! + ... + x^n/n!

sin x = x - x^3/3! + x^5/5! - ...

cos x = 1 - x^2/2! + x^4/4! - ...
