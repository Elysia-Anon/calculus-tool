考研数学核心公式库

专为考研数学一、二、三考生打造的极简速查表。本页收录了同济高数中最高频的等价无穷小、泰勒展开公式、各类积分表及易错点总结。

1. 常用等价无穷小 (x→0)

sin x ~ x
tan x ~ x
arcsin x ~ x
ln(1+x) ~ x
ex - 1 ~ x
1 - cos x ~ x2/2
(1+x)α - 1 ~ αx
ax - 1 ~ x·ln a

2. 必背泰勒公式 (x→0)

sin x x - x3/3! + o(x3)
cos x 1 - x2/2! + x4/4!
ex 1 + x + x2/2! + ...
ln(1+x) x - x2/2 + x3/3
1/(1-x) 1 + x + x2 + x3...
(1+x)α 1 + αx + α(α-1)x2/2!

3. 核心求导公式

(xμ)' μxμ-1
(ln x)' 1/x
(ax)' axln a
(sec x)' sec x·tan x
(csc x)' -csc x·cot x
(arcsin x)' 1/√(1-x2)
(arctan x)' 1/(1+x2)

4. 常用不定积分

∫ tan x dx -ln|cos x| + C
∫ cot x dx ln|sin x| + C
∫ sec2x dx tan x + C
∫ 1/(a2+x2) (1/a)arctan(x/a)
∫ 1/√(a2-x2) arcsin(x/a)
∫ 1/√(x2±a2) ln|x+√(x2±a2)|

5. 必备三角变换

sin2x + cos2x 1
1 + tan2x sec2x
sin 2x 2sin x cos x
cos 2x 2cos2x - 1
降幂 sin2x (1-cos 2x)/2
降幂 cos2x (1+cos 2x)/2

⚠️ 高数易错点总结

不定积分 千万别忘了 + C
定积分换元 上下限必须跟着换
洛必达法则 必须是 0/0 或 ∞/∞
奇函数对称区间 积分结果直接为 0
拐点 是点(x,y)，不是横坐标
